No. This workshop is designed for complete beginners. If you can use a browser and type, you can follow along.

We'll use AI tools (Claude/ChatGPT), a free code editor, and GitHub Pages for deployment. All free, all browser-based — nothing to install beforehand.

A practical, shareable web tool — details revealed on the day. It's designed so that everyone can follow along regardless of their background.
